# Solarized Osaka for Zed

![screenshot](./screenshots/solarized-osaka.png)

A Zed theme inspired by [Solarized Osaka](https://github.com/craftzdog/solarized-osaka.nvim).

This theme is created using [Zed's theme_importer](https://github.com/zed-industries/zed/tree/main/crates/theme_importer) tool, based on the [Solarized Osaka VSCode theme](https://marketplace.visualstudio.com/items?itemName=sherloach.solarized-osaka).
